DysonSphere
===========

Движок

20150607 Обновление движка. На текущий момент движок умеет выводить спрайты,
рисовать линии, запускать 1 выбранный модуль и создавать сервер/клиент 
и передавать между ними информацию

Инструкция
1 скомпилируйте проект
2 перейдите в каталог ../_bin
3 запустите Engine.exe - по умолчанию запустится модуль ZChatTest.dll для тестирования клиент-сервеного приложения
4 запустите SettingsEditor.exe для настроек движка
5 задайте Запускаемый модуль, Визуализацию и Устройство ввода
6 сохраните настройки в файл EngineSettings
7 запускайте Engine.exe
